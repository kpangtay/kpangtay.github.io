Coding Practice Platform

Galacticode is a platform for practicing foundational JavaScript concepts in a 3D gaming interface that guides players through increasingly difficult coding challenges. Players help Nova, a lost alien in space, get home by solving coding challenges to gain fuel points and unlock planets, all while practicing their JavaScript skills. Winner: Fullstack Academy's "Facebook Live Award"
