An e-commerce site created for a fictional reman shop designed and developed in a week using an agile workflow.

As a team we decided on the overall look and feel of the e-commerce site. We wanted to model it after other food ordering sites that also had a customization feature. So after some research we settled on the interface and tested the user flow to make sure it was efficient and easy to use for the consumer. The bowl customization was an interesting problem to think through. But as the lead designer i was able to methodically think through all the options and limitations we would have to implement to make sure it worked well. Uses sequelize many to many relationships and class and instance methods.

Technologies used react, redux, sequelize, express, node, and bootstrap.
